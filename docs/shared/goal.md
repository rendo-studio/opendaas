# Project Goal

VibeCoding 的长期目标已经固定到共享控制面，并作为后续计划、任务与状态的最高优先级锚点。

## 最终目标

Unspecified end goal

VibeCoding has not defined a long-lived end goal yet. Use `opendaas goal set` when the target outcome becomes clear.

## 背景与理由

当前项目已经进入正式推进阶段，因此需要把最终目标固定到共享文档包中，避免后续目标漂移。

## 完成标准

- Define the long-lived end goal for VibeCoding.
- Turn the current workspace into an explicit plan and task tree once the destination is clear.
- VibeCoding keeps shared docs and structured workspace state aligned while the project shape is still being discovered.

## 明确不做什么

- public hosted docs platform
- full SaaS control plane
- multi-agent orchestration
- cloud sync

共享层只固定长期目标本身；更细的执行状态不需要作为 authored docs 真相单独持久化。
